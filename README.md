## 🚀 Inicio Rápido

### Requisitos del Sistema

| Elemento | Requisito |
|----------|------------|
| **Sistema operativo** | macOS 13.0+ |
| **Hardware** | Apple Silicon (M1/M2/M3/M4) |
| **Python** | 3.11+ |
| **Memoria** | 16GB+ (Recomendado) |
| **Espacio en disco** | 5GB+ (Incluyendo Modelos) |

### Instalación con un Clic

```bash
# 1. Elegir directorio de instalación
cd ~/Downloads

# 2. Clonar proyecto
git clone https://github.com/matica0902/MLX-Video-OCR-DeepSeek-Apple-Silicon.git
cd MLX-Video-OCR-DeepSeek-Apple-Silicon

# 3. Inicio con un clic
./start.sh
```

La aplicación estará disponible en: **http://localhost:5000**

### Instalación Manual

```bash
cd ~/MLX-Video-OCR-DeepSeek-Apple-Silicon
curl -LsSf https://astral.sh/uv/install.sh | sh
uv venv
source .venv/bin/activate
uv pip install -r requirements.txt
python3 app.py
```
